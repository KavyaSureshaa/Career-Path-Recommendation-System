# Carrier-Path-Recommendation-System
Career Path Recommendation System is a machine learning-based application that predicts suitable career domains using user skills, education, and experience. It uses TF-IDF for text processing and LinearSVC for classification, and provides career growth paths through an interactive Streamlit interface
#  Career Path Recommendation System

## Project Overview
This project recommends career paths based on user skills, education, and experience using Machine Learning.

## Features
- Career domain prediction using LinearSVC
- Skill-based recommendation
- Career growth path visualization
- Streamlit interactive UI

##  Technologies Used
- Python
- Scikit-learn
- TF-IDF
- LinearSVC
- Streamlit

##  Dataset
Naukri.com Jobs Dataset

##  How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
